# 👁️ OptiView

OptiView is a web-based application designed to display e-commerce shop for glasses with admin area included 

It focuses on simplicity, performance, and a clean user experience.

---

## 🚀 Features

- 📊 Interactive and dynamic interface
- ⚡ Fast and lightweight
- 📱 Responsive design (mobile-friendly)
- 🧩 Modular and easy to extend
- 🎯 User-focused UI/UX

---

## 🛠️ Tech Stack

- HTML5 bootstrap 5 
- CSS3 (Flexbox / Grid)
- JavaScript (Vanilla JS /jquery /Ajax)
- Symfony /Easy admin
- three js + Augmented reality try on for glasses
